# Git DevOps Workflow Project

## Objective

This project demonstrates Git version control best practices in a DevOps environment.

## Tools Used

* Git
* GitHub

## Branch Strategy

* main
* dev
* feature/readme-update

## Tasks Performed

* Initialized repository
* Created development branch
* Created feature branch
* Updated project documentation
* Used pull request workflow
* Added .gitignore
* Created version tag

## Outcome

Learned Git workflows, branching strategies, pull requests, version control, and repository management.
